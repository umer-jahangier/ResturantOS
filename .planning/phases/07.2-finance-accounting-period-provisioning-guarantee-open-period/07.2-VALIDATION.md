## Sampling Rate

- **After every task commit:** Run `mvn -pl <touched-service> test` (unit-level only, fast)
- **After every plan wave:** Run `mvn -pl services/finance-service,services/platform-admin-service,services/auth-service -am verify`
- **Before `/gsd-verify-work`:** Full suite must be green, PLUS a live restart-and-manual-verify pass against the dev stack (a code fix in a jar that isn't running is not a fix — see Pitfall 3 in RESEARCH.md)
- **Max feedback latency:** ~360 seconds (full IT suite)
