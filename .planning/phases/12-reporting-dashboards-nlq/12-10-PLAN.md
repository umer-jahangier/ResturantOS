4. **THE NEGATIVE SECURITY CONTROLS.** These cannot be driven through the natural-language input (Claude will not reliably emit `DROP TABLE` on request, and we should not try to make it). Asserting only the 400 proves the API said no; asserting the table survived proves the database was never even asked.
