## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client → /api/v1/inventory/receipts | Manager receipt input (qty, cost, expiry) crosses into stock + valuation |
| finance-service → /internal/grn/pending-count | Service-to-service internal call, JWT-exempt, secret-header authenticated |
| concurrent receipts/depletions → stock row | MAC read-modify-write must not tear |
